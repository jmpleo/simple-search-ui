GET: /api/v1/search?q=&t=&p=&mm=&l=

Поиск с параметрами как на вкладке /search

  • q: query string
  • t: table
  • p: portion (page)
  • mm: max matches (max search results)
  • l: limit of portion (page)

GET: /api/v1/unloading/start?t=&q=

Запуск задачи на загрузку в фоновом режиме

  • q: query string
  • t: table

GET: /api/v1/unloading/status/{task_id}

Проверка статуса задачи

  • task_id: id задачи, возращаемый в методе /api/v1/unloading/status

GET: /api/v1/unloading/data/{task_id}.{ext}

Выгрузка готовых задач

  • task_id: id задачи, возращаемый в методе /api/v1/unloading/status
  • ext: формат выгружаемого файла. (json|csv)
